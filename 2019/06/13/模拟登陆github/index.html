会话和Cookies

会话和Cookies是用于保持HTTP连接状态的技术。
会话在服务端，用来保存用户的会话信息; Cookies在客户端，携带了会话ID信息。
会话维持：
当客户端第一次请求服务器时，服务器将自动创建一个会话并返回一个响应头中带有Set-Cookie字段的响应给客户端，用于标记是哪一个用户，客户端浏览器会把Cookies保存起来。
当浏览器下一次再请求该网站时，会把Cookies放到请求头发送给服务器，服务器通过识别Cookies即可找到对应的会话，从而判断用户是否是登录状态，然后返回对应的响应。
